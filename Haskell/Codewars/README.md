# Setup

Codewar has a [page](https://docs.codewars.com/languages/haskell) documenting the environment they use to run Haskell. 

## GHC Version

At the time of writing (2026-07-22), the version is GHC 9.2.5
Correspondingly, the matching language server version is HLS 1.9.1.0

Alternatively, GHC 9.2.8 and HLS 2.9.0.0 seemed to work

## Dependencies

Most of the dependencies should be in the `.cabal` file. A few packages with `codewars` in the name are not included because those are not published on Hackage
